# Install dbt Core 2.0

Local developmentⓘ

This page is for installing dbt Core 2.0, the Apache 2.0 open-source distribution of v2. To view the installation instructions, select **v2** from the version picker in the docs navigation. To install dbt Core v1.x, refer to [Install dbt Core v1.x](https://docs.getdbt.com/docs/local/install-dbt.md?version=1.12\&name=Core).

To upgrade an existing v1 project to v2, refer to [Upgrade to v2](https://docs.getdbt.com/docs/dbt-versions/core-upgrade/upgrading-to-v2.md?version=2.0).

To understand the differences between dbt Core v1.x, dbt Core 2.0, and Fusion, refer to [dbt licensing](https://docs.getdbt.com/docs/dbt-licensing.md?version=2.0).
