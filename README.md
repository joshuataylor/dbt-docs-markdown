# dbt-docs-markdown

The [dbt docs](https://github.com/dbt-labs/docs.getdbt.com) site uses Docusaurus, which offers `.md` files on their docs site, but there doesn't seem to be a easy-to-use version that is pre-compiled for use in tools like IDE Documentation (which Datamancer uses to show documentation for dbt functions in JetBrains IDEs such as PyCharm).
